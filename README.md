ReourceManagement excel addin is less about the resource management and more about some of the underly technology concepts that the addin exposes. Those key concepts are:
1) data storage in an external database. SQLite is used for this lighweight distributable example.
2) customXML to store the applied rules for a specific workbook, i.e. live with the workbook.
3) task pane for defining rules and applying them excel worksheet cells, related to 2) customXML.
4) cell based dropdown lists (single and multi select) tied to application rule.
5) Excel formula based use of rules to create spill ranges.
